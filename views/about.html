About

OpenPath enables place-based learning. It uses location-based videochat sessions to connect learners to events at museums, libraries, parks, makerspaces, and more.

FAQ

Is it free to use?

Yes. OpenPath is free to use.

How do I sign up?

Just login using Persona. Persona is a Web identity system designed by Mozilla. It allows you to sign in to Web sites with an email you choose without all the profile information associated with other login authentication services. You can add your profile information later on OpenPath for a more personalized experience.

What browser do I need?

The Google Chrome browser has all of the technology you need built-in. The Chrome browser is available for free here and is available for Windows, Mac and Linux. Firefox is not far behind and we plan to support that browser as well. For users of Internet Explorer, use the Chrome Frame. The technology is still maturing, so we expect most browsers will eventually support WebRTC fully.

Does this work on mobile phones?

If you have an Android-based phone, download the latest Chrome browser. The newest browsers allow WebRTC to work. If you have an iOS device, stay tuned. The WebRTC folks are working with the Webkit folks, but it is unclear when it will be available.

How does this work?

We are using a combination of HTML5, Javascript, and WebRTC. WebRTC stands for Web “Real-Time Communication.” The World Wide Web Consortium (W3C) is drafting an API definition to enable browser-based voice calling, video chat and P2P file sharing without plugins. The WebRTC initiative is a project supported by Google, Mozilla, and Opera.

What is a path?

We all have our own interest-driven learning journey to take. Over time, your individual path is the collection of events, places, and experiences in which you’ve participated.

On OpenPath, in addition to your profile, you can create events and paths. Events are things like an online tour you are giving of your favorite neighborhood. These can take place via our location-based video.

Paths are curated collections of interesting locations and events around a topic. For instance, a walking tour of a city, or a tour of multiple museums that include artwork by a particular artist. In the future, you will be able to upload your own paths that other people can use.

Can I just use it for videochat?

Sure you can. You can use it to hang out and share things with others. The emphasis of the site is on learning.
